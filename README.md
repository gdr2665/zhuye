# Frontend

## 项目技术

基于 React，并且使用 TypeScript。在库层面，本项目使用 Tencent TDesign，Ace Editor，Axios 等。

## 项目结构

```
src/
    App.tsx：主页
    components/
        answered：问答展示模块用户回答主组件，用例见 Problem
        aske：问答展示模块提问信息展示主组件，用例见 Problem
    pages/
        Ask：输入和提交问题的界面
        Explore：探索界面，暂未规划
        Login：登陆界面
        Problem：问答展示模块页面
        Register：注册界面
        Report：反馈页面
        Square：广场，即首页
        User：用户中心页
```
